CARA PASANG LOGO SENDIRI
========================

1. Siapkan file logomu dengan nama: logo.png
   (boleh juga: logo.jpg, logo.jpeg, logo.svg, atau logo.webp)

2. Drag/copy file logo itu ke folder ini (public/images/).

3. Restart server: tekan Ctrl+C di CMD, lalu jalankan "npm start" lagi.

4. Refresh browser. Emoji 🏠 di navbar otomatis berganti dengan logomu.

5. Kalau mau balik ke emoji rumah: hapus aja file logo.png dari folder ini.

REKOMENDASI UKURAN LOGO:
- Format: PNG dengan background transparan (paling oke)
- Ukuran: persegi (1:1), minimal 100x100 piksel
- Maks: 500x500 piksel (biar nggak berat)
